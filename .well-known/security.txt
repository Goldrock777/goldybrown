# If you have discovered a security issue with goldybrown.com or any
# of our connected services, please email security@goldybrown.com.
# We respond within 72 hours and credit responsible disclosures
# (with permission) on our /press/ page.
#
# Out of scope:
#   - Reports about missing security headers without a working exploit
#   - Self-XSS (must require user interaction beyond clicking a link)
#   - Outdated browser issues
#   - Generic vulnerability scanner output without analysis
#
# In scope:
#   - Authentication / authorisation flaws
#   - Server-side injection (SQLi, command injection, SSRF)
#   - Sensitive data exposure
#   - Account takeover paths
#   - Stored XSS
